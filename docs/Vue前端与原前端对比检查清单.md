## 技术细节

### MediaSlider 组件特性

1. **横向滚动：** 使用 `overflow-x: auto` 和 `flex` 布局
2. **导航按钮：** 根据滚动位置自动显示/隐藏左右按钮
3. **平滑滚动：** 使用 `scroll-behavior: smooth`
4. **响应式：** 不同屏幕尺寸使用不同卡片宽度
5. **触摸支持：** 支持移动端触摸滚动

### 样式类对照

**原始 CSS 类：**
- `.items-list-outer.list-inline.list-slider` - 滑动列表容器
- `.items-list-wrap` - 列表包装器
- `.items-list` - 列表
- `.item-thumb` - 媒体项

**Vue 组件类：**
- `.media-slider-wrapper` - 滑动列表包装器
- `.items-list-wrap.slider-list` - 滑动列表容器
- `.items-list.slider-items` - 滑动列表项容器
- `.item-thumb.slider-item` - 滑动列表中的媒体项
